# CampusIR

CampusIR is a multilingual **Hybrid PDF RAG (Retrieval-Augmented Generation)** system designed for university policy searches. It allows users to query university documents and receive accurate, context-aware answers.

## 🛠️ Tech Stack
- **Backend:** Python / Google Colab
- **RAG Framework:** LangChain / LlamaIndex
- **Vector Database:** ChromaDB / FAISS
- **LLM Integration:** OpenAI API / HuggingFace

## 🚀 Key Features
- **Multilingual Support:** Handles queries in multiple languages.
- **Hybrid Retrieval:** Combines keyword search with semantic vector search for high accuracy.
- **Context-Aware Answers:** Extracts precise information directly from official university policy PDFs.
